# Leaderboards Archive

This folder contains the old `leaderboards.json` that are necessary for old game versions since the 2nd quarter of 2024 (24Q2).

## How to use

* Rename `../leaderboards.json` to `../leaderboards.json.bak`
* Find the proper file (*consider that it might be 1 older/earlier than what you think*) based off the DLC and date.
* Copy the choosen file to parent directory `../`
* Rename such file to `leaderboards.json`.
